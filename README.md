### 比较流行的前端框架
我了解的比较典型的前端技术主要有：[ExtJS](http://www.sencha.com/products/extjs/), [jQuery](http://jquery.com/), [Bootstrap](https://github.com/twbs/bootstrap)。
我不太了解但是还算流行的技术有：[AngularJS](https://angularjs.org/), [KISSY](http://docs.kissyui.com/5.0/)。
